# Scripts Utilitas Aji Institute

Folder ini berisi script-script utilitas satu kali pakai untuk keperluan maintenance dan data management.

| Script | Fungsi |
|---|---|
| `clean_links.py` | Membersihkan URL / link yang rusak |
| `fix_thumbnail_color.py` | Memperbaiki warna thumbnail program |
| `rename_ajilanguage.py` | Rename slug/data AjiLangua |
| `inspect_db.py` | Inspeksi isi database SQLite |
| `make_slide.py` | Generate data slide/konten |
| `generate_ajistat_data.py` | Generate data dummy AjiStat |

> **Catatan:** Script-script ini dijalankan secara manual dari terminal, bukan bagian dari server production.
